◆ AcquirePage()

virtual PdfPage* PdfDoc::AcquirePage ( int  page_num)
pure virtual

Gets a PdfPage from a document. The page is cached, so that subsequent calls on the same PDPage return The same PdfPage and increments the reference count. The page remains in the cache as long as document exists or PdfPage::Release was not called. AcquirePage increments a pageï¿½s reference count. The page will not be released until the reference count is zero, or the application terminates. NOTE: After you are done using the page, release it using PdfPage::Release to release resources.

Parameters
page_numThe page number of the page to get. The first page is 0.
Returns
The requested page.
See also
PdfPage::Release
